# openwrt-packages
[openwrt-packages](https://github.com/wjz304/openwrt-packages)

## 使用
```
sed -i '$a src-git leesonaa https://github.com/leesonaa/openwrt-packages' feeds.conf.default
./scripts/feeds clean
./scripts/feeds update -a
./scripts/feeds install -a
make menuconfig
```
